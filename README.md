# Frontend Template (React.js)

### Instructions:
- Clone this repository:  
`http://eatlbd.net/banbeis/frontend-template.git`
- Go to the folder directory  
`cd frontend-template`  
- Copy `.env.dev` to `.env`  
- Update `.env` with required information
- Install dependencies  
`npm install` 
- Run application    
`npm start`
